The headset ceiling

Wearable AR hit a ceiling. The display has to leave the body. The room is always on.

We built the headset future and watched people take it off, every time. The face was always the limit, so we removed it. The tech to replace it only landed this year: scene understanding, on-device inference, bright short-throw optics. The window is open now.
